# Hackathon Submission Draft

## Project Title
SophieCare Sentinel

## Short Description
Multi-agent AI surgical workflow copilot where specialized agents coordinate intake, risk analysis, planning, compliance, and documentation.

## Long Description
SophieCare Sentinel is an enterprise multi-agent workflow prototype designed for surgical preparation environments. The system coordinates specialized AI agents that exchange context, analyze patient and procedure information, identify workflow-level risks, generate readiness checklists, validate missing compliance items, and draft structured documentation.

A Supervisor Agent orchestrates the entire process and makes the agent collaboration visible through a real-time command center. The demo highlights how collaborative AI systems can improve traceability, reduce fragmented workflows, support structured documentation, and keep humans in control of high-stakes operational decisions.

This prototype is not a medical device and does not provide diagnosis or treatment recommendations. It demonstrates an enterprise workflow pattern for human-reviewed surgical preparation support.

## Technology Tags
AI Agents, Multi-Agent Systems, Healthcare AI, Enterprise Workflow, Surgical Workflow, Next.js, TypeScript, Vercel, Gemini, OpenRouter, Ollama

## Demo Script
1. Introduce the problem: surgical teams handle fragmented information, manual checks, and time-sensitive documentation.
2. Load a sample patient case into SophieCare Sentinel.
3. Run the multi-agent workflow.
4. Show how the Supervisor Agent coordinates Intake, Risk, Planning, Compliance, and Documentation agents.
5. Show the final risk score, readiness score, checklist, missing items, and generated documentation.
6. Close with the value: visible agent collaboration, traceability, faster preparation, and human-in-the-loop safety.
